# 腾讯会议 Rooms 问题排障知识库

> 从 IMA 知识库自动同步，每周一更新。

## 目录结构

- `notes/` — 笔记原文（Markdown 格式）
- `files/` — 二进制文件（PPT、PDF 等）
- `manifest.json` — 知识库索引（包含所有条目的元数据）

## 内容说明

本仓库包含腾讯会议 Rooms 产品相关的技术支持知识：
- Rooms 工单故障记录
- 客户端已修复问题
- 网络问题排查
- 音频问题排查
- 云录制功能
-Rooms功能说明
- 通用问题（进退会、弹窗、UDP/QUIC 原理等）
- 产品培训材料（PPT）

## 同步方式

由 `sync_kb_to_git.py` 脚本每周一自动从 IMA 知识库同步。
